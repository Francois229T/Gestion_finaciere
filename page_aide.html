Foire Aux Questions (FAQ)

Comment créer une nouvelle activité ?

Pour créer une nouvelle activité, cliquez sur "Activités" dans le menu de navigation principal, puis sélectionnez "Créer une activité". Remplissez ensuite le formulaire avec les détails requis.

Comment puis-je visualiser mon solde total ?

Votre solde total est affiché sur le "Tableau de bord financier", accessible via le menu "Tableaux de bord".

Puis-je modifier une activité existante ?

Oui, allez dans "Activités" > "Gérer les activités", trouvez l'activité que vous souhaitez modifier dans le tableau, puis cliquez sur le bouton "Modifier" dans la colonne "Actions".

Besoin d'aide supplémentaire ?
